# How to Push to GitHub

Your code is ready to push to: **https://github.com/durrantkeith/triple-waza-friendship-challenge**

## Quick Push Instructions

From your local machine where you have this project:

```bash
# Navigate to the project directory
cd /path/to/triple-waza-friendship-challenge

# Add the GitHub remote (if not already added)
git remote add origin https://github.com/durrantkeith/triple-waza-friendship-challenge.git

# Push to GitHub
git push -u origin main
```

## Alternative: Using GitHub CLI

If you have GitHub CLI installed:

```bash
gh auth login
git remote add origin https://github.com/durrantkeith/triple-waza-friendship-challenge.git
git push -u origin main
```

## Alternative: GitHub Desktop

1. Open GitHub Desktop
2. Click "Add an Existing Repository"
3. Select this project folder
4. Click "Publish repository"
5. Choose "durrantkeith/triple-waza-friendship-challenge"
